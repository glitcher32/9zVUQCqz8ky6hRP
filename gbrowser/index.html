G-Browser is a light-weight Android browser that contains no trackers and doesn't require the need of Google services.

By default, the search engine and search suggestions are set to Google. I would highly recommend you change it to something different like DuckDuckGo if you care about your privacy. You can change this by going into the G-Browser settings then General Settings.

Yes, there are some bugs in the app that need to be fixed. For example, some hyperlinks will crash the app.

Permissions
★ permission.INTERNET: Needed to access the internet for browsing
★ permission.WRITE_EXTERNAL_STORAGE: Needed for downloading files and backing up data
★ permission.READ_EXTERNAL_STORAGE: Needed for downloading files and restoring data
★ permission.ACCESS_FINE_LOCATION: Needed for sites that want your location (you can deny location access)
